1
📊 Generate Excel from Month Plan

Open Month Plan and navigate to the week you want to review. It displays all your scheduled events for that week. Click Export to Excel (.xlsx) — this generates a structured spreadsheet with your time blocks, categories, and event details that the AI can read.

💡 Tip: Make sure the correct week is selected in Month Plan before exporting — the date in the Excel filename should match the week you're reviewing here.
2
🤖 Import Excel → AI generates JSON

Open Horizon Generator and import the Excel file you just exported. Claude AI analyses your week — categorising time, identifying flow/drain moments, extracting wins and insights — and produces a structured JSON file that maps directly to every field in this Week Review.

💡 Tip: Review the AI's output before exporting. You can edit any field in the generator before saving the JSON — the AI gives you a strong first draft, not a final answer.
3
📋 Import JSON → Complete your Review

Back here in Week Review, click the cloud menu ☁ → Import from file and select the JSON. All fields are pre-filled instantly. Add your personal reflections, run the AI Coach if you want deeper insight, then click Save Review (and optionally Save to Cloud to sync across devices).

💡 Tip: The JSON import fills the AI-generated fields. Your personal Reflections, Coach question, and 30-Year Vision are always yours to write — the AI sets the stage, you write the meaning.
☁
Cloud Sync (optional but recommended)

Sign in with Google via the ☁ Sign in button in the top-right. Once signed in, use Save to Cloud after each review and Load from Cloud to restore on any device. Enable Auto-sync to have changes saved automatically every few seconds.
