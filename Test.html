🗓 Connect Google Calendar

You need a free Google Cloud OAuth Client ID. This takes about 5 minutes to set up once.

  1. Go to console.cloud.google.com
  2. Create a project → Enable the Google Calendar API
  3. Go to Credentials → Create OAuth 2.0 Client ID → Web application
  4. Add this page's origin to Authorised JavaScript origins:
  5. Copy the Client ID and paste below
