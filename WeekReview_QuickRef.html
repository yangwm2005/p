Weekly Review

Trigger
Every Saturday
Owner
Dr. Wen-Ming Yang · WIG
1
Export
Get the week's events from Monthplan
Open monthplan-v24.html → collect all events → export as Excel (.xlsx).
URL: yangwm2005.github.io/p/monthplan-v24.html
2
Primary Analysis
Drop Excel + prompt file into DeepSeek
Upload the Excel file together with Weekly Event Analysis Deepseek v3.txt.
Ask for: personal progress, issues, and a full weekly summary.
Save output — bring to WIG meeting for reference.
3
Secondary Analysis
Repeat in Gemini for a second perspective
Same Excel + same prompt → Gemini.
Output is less structured; treat as supplementary. DeepSeek remains primary.
4
Archive
Add analysis to NotebookLM monthly collection
Compile this week's AI outputs into the month's NotebookLM PDF.
Builds a searchable long-term database for monthly pattern analysis.
※
Key principle: The quality of this review is bounded by the quality of records kept in Monthplan throughout the week. Consistent, detailed event logging is the foundation — everything else follows from it.
